# Opportune v0.1.1 Public Alpha

Opportune 0.1.1 consolidates the local-first Version 1 product into an
installable public-alpha candidate: approval-gated onboarding, deterministic
job discovery and ranking, a packaged local dashboard, platform-aware state,
privacy controls, and notification-only update checks.

Before upgrading, run `opportune privacy backup --json`. Install only the wheel
and checksum attached to this release. Opportune never updates itself silently.

Known limitations and exact evidence are maintained in `RELEASE_SCORECARD.md`.
Email integration, outreach, application automation, and other Version 2
features remain intentionally excluded.
